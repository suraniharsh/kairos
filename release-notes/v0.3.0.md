## What's New

- Add universal install scripts for Linux, macOS, and Windows

## Bug Fixes

- Fix Cloudflare custom domain route (remove wildcard that caused routing issues)

## Install

**Linux / macOS**
```bash
curl -fsSL https://kairos.suraniharsh.com/install.sh | bash
```

**Windows (PowerShell)**
```powershell
irm https://kairos.suraniharsh.com/install.ps1 | iex
```
