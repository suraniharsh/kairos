## What's New

- Add a version notifier: kairos now checks for newer releases in the background and surfaces a hint when one is available (closes #18)
  - **TUI**: `↑ <version>` hint in the status bar
  - **CLI**: one-shot commands print a short notice to stderr, sourced from the cached check (no extra network call)
  - Result is cached for 24h under `$XDG_CACHE_HOME/kairos/latest_version.json`; set `KAIROS_NO_UPDATE_CHECK=1` to disable
- Add a `GET /version` route to the `kairos.suraniharsh.com` Cloudflare Worker, proxying and edge-caching the GitHub releases API so update checks don't hit GitHub's per-IP rate limit directly

## Install

**Linux / macOS**
```bash
curl -fsSL https://kairos.suraniharsh.com/install.sh | bash
```

**Windows (PowerShell)**
```powershell
irm https://kairos.suraniharsh.com/install.ps1 | iex
```
